---

## CSS Techniques Used

### ✅ Flexbox

Used for:

* Navigation alignment
* Card layouts
* Responsive positioning

Example:

```css
display: flex;
justify-content: space-between;
align-items: center;
```

---

### ✅ CSS Grid

Used to create responsive project/service layouts.

Example:

```css
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
```

---

### ✅ Media Queries

Ensures responsiveness on all devices.

Example:

```css
@media (max-width: 768px) {
    .navbar {
        flex-direction: column;
    }
}
```

---

### ✅ Hover Effects & Transitions

Used for better user interaction.

Example:

```css
transition: 0.3s ease;
transform: scale(1.05);
```

---

### ✅ Animations

CSS animations improve modern UI feel and engagement.

Used for:

* Fade-in effects
* Hover interactions
* Smooth transitions

# ⚡ JavaScript Functionality

JavaScript adds interactivity and dynamic behavior.

---

## Features Implemented Using JavaScript

### ✅ Mobile Navigation Menu

```javascript
menuBtn.addEventListener("click", () => {
    nav.classList.toggle("active");
});
```

Purpose:

* Opens/closes mobile menu
* Improves mobile responsiveness

---

### ✅ Smooth Scrolling

```javascript
document.querySelectorAll('a[href^="#"]').forEach(anchor => {
    anchor.addEventListener('click', function(e) {
        e.preventDefault();
    });
});
```

Purpose:

* Smooth navigation between sections

---

### ✅ Dynamic Interactions

JavaScript is used for:

* Button interactions
* Menu toggling
* UI responsiveness
* Animation triggering
